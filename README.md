# Spider Solitaire Mountfield

A Spider Solitaire game with Mountfield themed cards, built with Electron.

## Prerequisites

- [Node.js](https://nodejs.org/) (v16 or later)
- npm (comes with Node.js)

## Installation

```bash
npm install
```

## Running the App

```bash
npm start
```

This launches the game in an Electron window.

## Building a Distributable

| Platform | Command |
|----------|---------|
| Windows  | `npm run build:win` |
| macOS    | `npm run build:mac` |
| Linux    | `npm run build:linux` |
| All      | `npm run build` |

Built packages are output to the `dist/` folder.
